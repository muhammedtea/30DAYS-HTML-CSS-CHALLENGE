Ordered List of how to create a new repository on the command line.
  1. echo "#resipotory name"
  2. git init
  3. git add README.md
  4. git commit -m "first commit"
  5. git remote add origin
  6. push -u origin main
Unordered list on how to push an existing repository from the command line
  • git remote add origin
  • git branch -M main
  • git push -u origin main
